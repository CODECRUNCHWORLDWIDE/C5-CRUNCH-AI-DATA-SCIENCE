# Week 3 — Challenges

One challenge this week. Optional, but it is the closest thing in the curriculum to "what a data journalist actually does on a Tuesday": pick a published chart, look at it carefully, and recreate it from scratch with synthetic data. The recreation is the easy part; the critique is the point.

1. **[Challenge 1 — Recreate an FT chart](challenge-01-recreate-an-FT-chart.md)** — pick one chart from the Financial Times' [Chart Doctor](https://github.com/Financial-Times/chart-doctor) gallery or any FT data story you can read for free. Reproduce it with matplotlib and synthetic data, then write a one-page critique of the original. (~2 hours)

The challenge sets up the mini-project: the mini-project asks for three of these and a deeper critique each. Doing one carefully now means the mini-project feels like an extension of the same habit.
