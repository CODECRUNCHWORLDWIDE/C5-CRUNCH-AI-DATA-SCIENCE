# Week 1 — Challenges

One challenge this week. It is optional and transformational. Skip it and you will know NumPy well; do it and image processing stops feeling like magic.

1. **[Challenge 1 — Image filter, pure NumPy](challenge-01-image-filter-pure-numpy.md)** — implement grayscale, a 3×3 box blur, and Sobel edge detection on a real image using only NumPy. No PIL, no OpenCV. (~2–3 hours)

This challenge is the technical foundation of the [Week 1 mini-project](../mini-project/README.md). If you do the challenge first, the mini-project is mostly about packaging and writing up your work.
